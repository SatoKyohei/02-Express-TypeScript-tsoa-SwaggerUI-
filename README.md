## 実行したコマンド（順不同）
- npm init -y
- npm i tsoa express typescript @types/node @types/express
- npm i nodemon ts-node concurrently
- npm i swagger-ui-express @types/swagger-ui-express
- npm run dev
  - コード変更の自動反映とAPI仕様書の自動生成
- npm run build
  - API仕様書の生成とTypeScriptをJavaScriptにコンパイル
